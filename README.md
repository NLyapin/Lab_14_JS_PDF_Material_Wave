# Лабораторная работа №14

Реализована интерактивная страница резюме на чистых HTML/CSS/JS.

## Что сделано

1. Сверстана страница резюме.
2. Все текстовые блоки редактируются на месте через `contenteditable`.
3. Изменения автоматически сохраняются в `localStorage` и восстанавливаются после перезагрузки.
4. Добавлены анимации при изменении текста.
5. Добавлен эффект `Material Wave` (ripple) при кликах.
6. Реализована кнопка `Скачать PDF` через печать страницы (`window.print()`) с отдельными print-стилями.
7. Добавлена адаптивность для мобильных устройств.

## Стек

- HTML
- CSS
- JavaScript (без библиотек и фреймворков)

## Запуск локально

Проект статический, поэтому можно открыть `index.html` напрямую в браузере.

Либо через Vite:

```bash
npm install
npm run dev
```
